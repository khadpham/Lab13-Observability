Observability Flow

Use this diagram to explain how one request turns into logs, traces, and metrics.

1. Request

User traffic hits the FastAPI app and carries request body, headers, and feature context.

2. Middleware

Correlation ID is injected or propagated so every request can be followed across evidence sources.

3. Logging

Structured JSON logs capture request context while PII is scrubbed before it reaches the log file.

4. Tracing

Langfuse records the agent run, input/output previews, latency, token usage, and incident clues.

5. Metrics

/metrics aggregates latency, traffic, errors, cost, tokens, and quality for dashboard panels.
